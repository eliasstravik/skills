# skills

Personal agent skills by Elias Stravik.

## Install

```bash
npx skills@latest add eliasstravik/skills --skill html
npx skills@latest add eliasstravik/skills --skill automake
```

## Skills

- [Automake](./skills/automake/SKILL.md): set up and run an automake improvement loop with git-backed candidate commits and LLM-as-judge keep/revert decisions.
- [HTML](./skills/html/SKILL.md): create complete, locally served HTML artifacts for visual specs, decks, diagrams, illustrations, interactive reports, prototypes, explainers, and custom editors.
