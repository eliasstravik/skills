# Approval inbox idea

Northstar Forms is a fictional internal request tool. Teams want one inbox for
pending access requests that currently arrive through email.

Known facts:

- Every request already has requester, resource, reason, submitted time, and
  one assigned approver.
- Version one may approve or reject one request at a time.
- Bulk action, delegation, escalation, and notification changes are not yet
  agreed.
- The plan must not assume a mobile app or a new backend.
